Selenium, c'est quoi ?

Selenium est un framework de tests informatiques développé en Java mais qui propose certains outils pour pouvoir être utilisé avec différents langages comme Python ou PHP. Il permet d'intéragir avec différents navigateurs Web (Google Chrome, Mozilla Firefox).
C’est un outil de test dit dynamique (à l’inverse des tests statiques qui ne nécessitent pas l’exécution du logiciel) ce qui facilite le test fonctionnel.


La version actuelle de Selenium utilise principalement le HTML et le JavaScript. Cela permet aux développeurs de tester et d'enregistrer les interactions avec une application Web.
Le principal intérêt est de pouvoir automatiser ces interactions afin de les répéter autant de fois que nécessaire. Les principales composantes de ce processus de test sont les suivantes :



Principaux composants

Selenium IDE

IDE (Integrated Development Environment) est l'outil que vous utilisez pour développer vos cas de test Selenium.
C'est une extension Chrome et Firefox facile à utiliser et c'est généralement le moyen le plus efficace de développer des cas de test.
Il enregistre pour vous les actions des utilisateurs dans le navigateur, en utilisant les commandes Selenium existantes, avec des paramètres définis par le contexte de cet élément.
Ce n'est pas seulement un gain de temps, mais aussi un excellent moyen d'apprendre la syntaxe du script Selenium.



Selenium WebDriver

Si vous commencez avec l'automatisation des tests de site Web de bureau ou de site Web mobile, vous allez utiliser les API WebDriver. WebDriver utilise les API d'automatisation du navigateur fournies par les fournisseurs de navigateurs pour contrôler le navigateur et exécuter des tests.
C'est comme si un vrai utilisateur utilisait le navigateur.
Aucune compilation n'est nécessaire.



Selenium Grid

Selenium Grid vous permet d'exécuter des cas de test sur différentes machines sur différentes plateformes.

Après le développement des tests WebDriver, vous devrez peut-être exécuter vos tests sur plusieurs combinaisons de navigateur et de système d'exploitation. C'est là que Selenium Grid est utile.
