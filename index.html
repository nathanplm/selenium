Selenium, c'est quoi ?

Selenium est un framework de tests informatiques développé en Java mais qui propose certains outils pour pouvoir être utilisé avec différents langages comme Python ou PHP. Il permet d'intéragir avec différents navigateurs Web (Google Chrome, Mozilla Firefox).
C’est un outil de test dit dynamique (à l’inverse des tests statiques qui ne nécessitent pas l’exécution du logiciel) ce qui facilite le test fonctionnel.


La version actuelle de Selenium utilise principalement le HTML et le JavaScript. Cela permet aux développeurs de tester et d'enregistrer les interactions avec une application Web.
