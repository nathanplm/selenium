Demo Selenium IDE

La démo suivante montre un exemple d'utilisation de Selenium IDE.

Elle montre comment enregistrer son écran et ainsi obtenir un script de test contenant toutes les actions qui ont été effectuées par l'utilisateur. Il est possible par la suite d'exporter ce script.

Voici le script généré au format JSON de ce script

Cette démo ne montre qu'une utilisation basique de Selenium IDE, en effet le script obtenu n'a aucun réel intérêt à exploiter.

Cependant, cela laisse présager de l'utilisation possible de cet IDE notamment avec la personnalisation des tests ou l'exportations dans un langage de programmation en particulier.

Elle permet également de voir les concepts comme les "localisateurs" ou la gestion des évènements.
