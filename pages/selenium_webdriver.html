Selenium WebDriver

Description :

Selenium WebDriver est une API compacte orientée object qui est conçue comme une interdace de programmation simplifiée.

Cela permet notamment de "piloter" un navigateur de manière native comme le ferait un utilisateur classique, soit localement, soit sur une machine distante utilisant le serveur Selenium.

Selenium WebDriver fait référence à la fois aux différents langages de programmation mais également aux implémentations du code de contrôle du navigateur utilisé.



Comment l'utiliser ?

Le framework Selenium relie tous ces éléments via une interface utilisateur qui permet aux différents backends de navigateur d'être utilisés de manière transparente, permettant une automatisation multi-navigateurs et multi-plateformes.

La configuration de Selenium est assez différente de la configuration d'autres outils commerciaux.



Pré-requis :

Avant de pouvoir commencer à écrire du code Selenium, vous devez installer les bibliothèques de liaisons de langue pour la langue de votre choix, le navigateur que vous souhaitez utiliser et le pilote de ce navigateur.

En résumé, avant d'utiliser Selenium WebDriver, vous devez :

  • Installer une bibliothèque Selenium
  • Installer les pilotes du navigateur
  • Utiliser le code Selenium pour ouvrir ou fermer un navigateur et interragir avec



Son fonctionnement :

Ouvrir / Fermer un navigateur

Un fois que vous avez choisi le navigateur que vous allez utiliser, vous avez la posibilité d'interragir avec celui-ci, notamment en l'ouvrant ou en le fermant.

Le cas ci-dessus utilise le navigateur Chrome et le langage de programmation Java.

Consulter le site officiel si vous voulez avoir le code pour utiliser d'autres langages ou navigateurs.

Naviguer sur une page web

A l'aide du code ci-dessous, il est possible de naviguer jusqu'à une page web spécifique.

Demander des informations sur le navigateur

Avec Selenium, il est possible de récupérer plusieurs informations sur le navigateur comme la taille de la fenêtre, sa position, les cookies ou le titre du site.

Etablir des stratégies d'attente

Selenium permet de répondre à un gros défi qui est la synchronisation du code avec l'état actuel du navigateur.

En fait, vous pouvez mettre en place des stratégies afin de vous assurer que l'élément se trouve sur la page avant d'essayer de le localiser et que l'élément est dans un état interactif avant d'essayer d'interagir avec lui par exemple.

Trouver un élément

Il est possible de trouver un élèment sur la page grâce à plusieurs "localisateurs" comme l'id, le name ou autre.

Agir sur un élément

Vous avez la possibilité d'agir sur un élèment. Il existe seulement 5 commandes basiques qui peuvent être utilisées sur un élèment :

  • click : pour cliquer sur un élèment
  • send keys : pour ajouter du texte à un champs ou à un élèment éditable
  • clear : pour vider un champs ou un élèment éditable
  • submit : seulement applicable pour un formulaire
  • select : pour sélectionner un élèment en particulier

Demander des informations sur un élément

Il est également possible de récupérer la valeur d'un élèment.



Selenium WebDriver permet évidemment de faire beaucoup plus de choses.
N'hésitez donc pas à consulter le site officiel pour plus d'informations.
